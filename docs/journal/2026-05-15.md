## Co se udělalo

Po dni od publikace (v0.3.1 živé na PyPI) jednodenní docs sprint —
zpřesnění licencí v README:

### Per-tool license tabulka

Místo jednoho generického "MIT" v README přidána tabulka rozlišující:
- **Software licence**: MasKIT/PONK/NameTag/UDPipe všechny MPL 2.0
  (commercial OK)
- **Model licence**: NameTag a UDPipe modely **CC BY-NC-SA**
  (non-commercial)
- **API služba**: TOS oddělené od licence software/modelů

Důležité pro komerční uživatele: i když je MPL 2.0 commercial-friendly,
modely (CC BY-NC-SA) pro komerční použití vyžadují kontakt s autory.

### Claude Desktop MS Store omezení

Nová MS Store verze Claude Desktop ("Cowork" UI) podporuje **pouze
remote MCP servery přes HTTP URL**. Lokální stdio servery jako `ufal-mcp`
zde **přidat nelze**. Workaround: Claude Code CLI nebo starší Claude
Desktop (.exe/.app installer).

### Untested klienti

OpenAI Codex CLI, Cursor, Windsurf, Cline, Zed, VS Code Copilot Agent
mají config snippety s "(autorem netestováno)" disclaimerem. Standardní
MCP stdio formát, teoreticky funguje — PR s potvrzením vítaný.
