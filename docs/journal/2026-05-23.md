## Lessons learned

- **Iterative bug hunting** — každý commit po opraveném bugu, mass
  corpus test po každé změně. Žádný "big bang" — důsledné
  per-bug-per-commit cyklus.
- **Test data integrita** — některé failury vypadají jako "test bugs",
  ale často jsou to reálné bugy. Honest re-audit nutný před vyhlášením
  100%.
- **Strict default ON** — pro production anonymizaci doporučeno
  `placeholder_mode=True` (deterministic OSOBA1/MESTO1 + dedup) +
  `strict=True` (NameTag pre-pass pro státní instituce).
- **International coverage** dosažitelná systematickým mapováním
  ISO standardů (IBAN, ISO 3166-1, ISO 13616, ISO 6346) + per-country
  national ID research.
- **CI flakes řešit retry, ne ignorovat** — `Wandalen/wretry.action`
  + `nick-fields/retry` na všechny external dependency steps.

## 12. ufal-mcp v0.7.30 — Defense-in-depth + adversariální normalize

Production-grade vyžaduje "trust but verify".

**Nové moduly:**

### `maskit_audit.py` — Output validation
- Post-pipeline scan výstupu na zbytkové PII
- 3 severity levels (critical/high/medium)
- `audit=True` warning, `strict_audit=True` → `ResidualPIILeak` exception
- Placeholder masking přes PUA `` aby `IBAN3` netriggeroval IBAN detector

### `maskit_normalize.py` — Adversariální defenses
- NFC unicode normalization (NFD `a + ◌́` → composed `á`)
- Zero-width chars strip (ZWSP/ZWNJ/ZWJ/BOM/MVS/SOFT HYPHEN)
- Bidi override strip (LRO/RLO/PDF/LRE/RLE/LRI/RLI/FSI/PDI/LRM/RLM)
- Non-Latin digit normalization — celá Unicode `Nd` kategorie
  (full-width CJK `１２３`, Arabic-Indic `٠١٢`, Persian `۰۱۲`, Devanagari
  `०१२`, Bengali, Tamil, Thai, Khmer, Myanmar, Tibetan)
