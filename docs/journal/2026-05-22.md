## Klíčové milníky

### v0.7.8 — 5 pre-pass regex patternů (22:22)

Test reálné smlouvy odhalil 5 nepokrytých PII formátů:
- Slovní datumy s všemi pádovými tvary měsíců
- UCET 7-10 digit context
- OP kontextové pattern
- PSČ standalone s currency blacklist
- SPZN sp. zn. formats (NS Cdo/Tdo, ÚS, NSS Afs/Azs)

### v0.7.9 — Range years (22:45)

Wikipedia discovery: `(1937–2020)` v závorkách jako rok narození–úmrtí
nebyl chytán. Nový pattern `\((\d{4}[–-]\d{4})\)` → DATUM placeholder.

### v0.7.10 — Post-process layer (22:57) ⭐

**Klíčový architekturní milník.** Místo přidávání další pre-pass logiky
zavedena **post-process vrstva** — opravuje výstup MasKITu po jeho
zpracování:

- **`revert_institutional_persons`** — `INSTITUCE1 STAT1` (GA ČR) →
  zpět na `GA ČR` přes preserve list lookup
- **`merge_compound_cities`** — "Lhoty za Červeným Kostelcem" jako

## Lessons learned

- **Wikipedia = brutální stress test** — reálná encyklopedie má víc
  edge cases než kdokoliv vymyslí synteticky.
- **Post-process layer** lepší než víc pre-pass logiky — pre-pass
  je rigidní, post-process může opravit MasKIT chyby s plným kontextem.
- **Preserve list nutný** — bez něj se anonymizuje i ICD-10, ISO,
  grantové kódy → useless output pro vědecké/medicínské použití.
- **Cross-sentence kontext nebezpečný** — institutional revert window
  musí respektovat sentence boundaries, jinak leak z předchozí věty.
